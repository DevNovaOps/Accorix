How Auto Analytical Model Works:

The model is applied if any one field matches the transaction line.

If multiple fields match, the model becomes more specific and takes priority.

Models with fewer matched fields are more generic, while more matches make them stricter.

This allows flexible yet prioritized automatic analytic model.

Example:
In the above model, the Product Category is Wooden Furniture and the Partner is Mr. A.
When the customer or vendor selected is Mr. A and the product belongs to the
Wooden Furniture category, this analytic distribution will be applied.
If no partner is selected, the distribution will still apply whenever the selected product
falls under the Wooden Furniture category.
Hence, more selected fields make the rule stricter, while fewer selections make it more generic.
